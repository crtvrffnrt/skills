# Big Tech Catalyst Prompt

Use this file together with [PROMPT_FRAMEWORK.md](./PROMPT_FRAMEWORK.md).

## Scope
Identify the big-tech stocks from the watchlist with the highest probability of significant movement within the next 24 hours:
- Alphabet Inc. (`GOOG`, `GOOGL`)
- Microsoft (`MSFT`)
- Amazon (`AMZN`)

## Priority Catalyst Types
- AI product launches; hyperscaler capex changes.
- Cloud demand shifts; antitrust/regulatory developments.
- Enterprise software demand; cybersecurity incidents.
- Data-center expansion; material filings; strategic partnerships.

## Sector-Specific Analysis
### Ecosystem Propagation
Assess how adjacent-company events change sentiment for the whole group.
- Microsoft capex expansion -> bullish cloud and AI read-through.
- AWS demand shift -> broader view on enterprise cloud strength.
- Alphabet AI launch -> sentiment change across search, cloud, and platform AI.

### Scoring Bias
Weight these catalysts more heavily:
- Regulatory or antitrust action; major cybersecurity incidents.
- Earnings surprise; hyperscaler infrastructure deals.
- Material SEC filings; cloud or platform demand shifts.

## Output Title
`Top 3 Big-Tech-Aktien mit der höchsten erwarteten Kursbewegung (24 Stunden)`
